Commands

Introduction

NetBSD, being a UNIX based operating system has many commands that are also available in Linux. Commands such as:

  • ls: list files
  • cd: change directory
  • pwd: present working directory
  • cat: concatenate files and print on standard output
  • cp: copy files and directories
are common in many UNIX based systems. Given below are some unique commands that are usually only found in NetBSD operating systems.

Unique Commands

Command Name Description Code
pkg_add Add a binary package 
pkg_add binary_package.tgz
pkg_info List all installed packages 
pkg_info
pkg_delete Delete the package 
pkg_delete the_package_name
pkg_tarup Create a binary package file from an already installed package 
pkg_tarup the_package_name
wsconsctl Get or set wscons state 
wsconsctl
